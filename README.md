apu_tt (tt)
===========

Display intake timetable Asia Pacific University with open web services API.

Features
--------
- pretty table
- fully colored
- cache response
- offline request

Building
--------
tt is written in [Rust](https://rustup.rs/).

    $ cargo install --git git://github.com/pickfire/apu_tt
