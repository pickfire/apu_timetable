apu_tt (tt)
===========

Display intake timetable Asia Pacific University with open web services API.

Features
--------
- pretty table
- WIP color
- WIP cache
- WIP config

Building
--------
tt is written in [Rust](https://rustup.rs/).

    $ git clone https://github.com/pickfire/apu_tt
    $ cd apu_tt
    $ cargo build --release
    $ ./target/release/tt
